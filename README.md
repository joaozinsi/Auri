# Auri

Landing/MVP de e-commerce de joias autorais, feito em React + Vite.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy no GitHub Pages

Este projeto está configurado para publicar em:

```text
https://SEU_USUARIO.github.io/Auri/
```

Se o repositório tiver outro nome, altere o campo `base` em `vite.config.js`.

No GitHub, ative:

```text
Settings > Pages > Source > GitHub Actions
```

Depois, cada push na branch `main` publica uma nova versão.
